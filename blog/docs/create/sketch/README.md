## 一、工具

### 铅笔的种类及运用

| 类型 | 2H   | HB   | 2B   | 4B   | 6B   | 8B   | 10B  | 12B  |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 类别 | 硬铅 | 硬铅 | 中铅 | 中铅 | 软铅 | 软铅 | 软铅 | 软铅 |

- H为hard，表示硬；B为black，表示黑
- 有单数B，但是没有必要使用
- 软铅颗粒大，不紧实，最黑；硬铅颗粒小，紧实，最不黑
